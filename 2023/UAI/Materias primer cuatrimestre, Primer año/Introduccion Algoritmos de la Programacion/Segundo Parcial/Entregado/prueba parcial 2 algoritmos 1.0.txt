/*
En una empresa se ingresan los datos de los 10 empleados de la siguiente manera:
		Codigo del empleado.
		Categotria (Puede ser del 1 al 4)
		Cantidad de años de antiguedad
		mes (1 a 12)

EL ingreso funaliza cuando se ingresa una categoria 0.

El sueldo basico segun la categoria:
   		Para categoria 1, 3000.
   		Para categoria 2, 2000.
   		Para categoria 3, 1000.
   		Para categoria 4, 500.

El sueldo es la suma del basico segun la categoria mas 100 por año de antiguedad.

Mostrar en pantalla
   		1. Mostrar por empleado los sueldos pagados por cada empleado.
   		2. Mostrar por mes los sueldos pagados por empleado.
   		3. mostrar el promedio de sueldos por mes.
   		4. Porcentaje que representa el sueldo anual de cada empleado sobre el total pagado.

*/

{

    printf("[4] Ver Sueldo por Mes por Empleado.\n");
    printf("[0] Cerrar menu.\n");
    printf("\n============================================\n");
    printf("Escribir Opcion: ");
    scanf("%d", &OpcionMenu);
}

void SwitchMenu()
{
    switch(OpcionMenu)

        case 0:
            return 0;
            break;

            IngresarCodigoEmpleado_1();

            break;
        case 2:
            IngresarCategoriaEmpleado_2();

            break;

            break;
        case 4:

            break;
    }
}
